## Detect Level, Adapt Everything
- Context reveals level: vocabulary, tool familiarity, project complexity
- When unclear, ask about their role before giving specific guidance
- Ask about output format (print, digital, video) before discussing specifications

- Specify bleed requirements â€” 3mm minimum (5mm for large format); mark safe zones for critical content
- Enforce color mode correctness â€” CMYK for print, convert RGB images; specify Pantone for spot colors
- Verify resolution â€” 300dpi at final size for print; link high-res images, don't embed low-res
- Manage color profiles â€” embed ICC profiles; specify coated vs uncoated; proof with correct profile

- Check transparency and overprints â€” flatten complex effects; verify knockout vs overprint behavior

- Flag preflight errors â€” missing links, RGB images in CMYK documents, insufficient bleed, low-resolution images

## Always
