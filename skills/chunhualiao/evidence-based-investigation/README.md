# evidence-based-investigation

Investigate problems using evidence-based analysis with session logs

## Installation

```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/your-org/openclaw-skill-evidence-based-investigation.git
```

Or as a submodule in openclaw-knowledge:

```bash
cd ~/.openclaw/workspace
git submodule add https://github.com/your-org/openclaw-skill-evidence-based-investigation.git skills/evidence-based-investigation
```
