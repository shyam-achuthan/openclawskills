# nanobanana pro 名媛拍照指南 - 妆容造型指导

## 一、底妆篇

### 核心关键词
- **清透奶油肌**：natural dewy skin, flawless complexion, porcelain skin
- **光泽感底妆**：luminous skin, healthy glow, radiant finish
- **零毛孔妆效**：poreless skin, airbrushed complexion

### AI提示词参考
```
flawless porcelain skin with natural glow, dewy foundation, luminous complexion, soft focus skin texture, healthy radiant skin, translucent makeup base
```

### 实用技巧
- 强调"半哑光+微光泽"的质感，避免过度油光
- 可添加"soft focus"或"airbrushed"描述柔焦效果
- 肤色描述：fair skin（白皙）、porcelain skin（瓷白）、peachy skin（蜜桃肌）

---

## 二、眼妆篇

### 1. 大地色系眼妆
**关键词**：warm brown eyeshadow, neutral earth tones, nude eye makeup
```
subtle brown eyeshadow, warm caramel tones, soft gradient eye makeup, natural smoky eyes
```

### 2. 桃花妆/粉色系
**关键词**：peach eyeshadow, rosy pink eyes, soft pink tones
```
delicate peach eyeshadow, soft pink eye makeup, romantic rosy lids, feminine pink tones
```

### 3. 睫毛与眼线
**关键词**：long eyelashes, winged eyeliner, defined lash line
```
long voluminous eyelashes, subtle cat-eye liner, perfectly curled lashes, defined upper lash line
```

### 实用技巧
- 名媛眼妆以"精致不夸张"为原则
- 强调睫毛的纤长卷翘感："feathery lashes"、"doll-like lashes"
- 眼线以细眼线为主，眼尾可微微上扬

---

## 三、眉形篇

### 1. 野生眉（推荐）
**关键词**：natural fluffy brows, bushy eyebrows, feathered brows
```
natural feathered eyebrows, soft fluffy brow texture, well-groomed bushy brows, defined brow shape with natural texture
```

### 2. 韩式平眉
**关键词**：straight brows, Korean-style eyebrows, soft straight brow
```
soft straight eyebrows, Korean-style straight brows, natural straight brow shape
```

### 3. 微挑眉
**关键词**：arched eyebrows, slightly lifted brows, elegant brow arch
```
elegant arched eyebrows, softly lifted brow tail, refined brow shape with gentle arch
```

### 实用技巧
- 野生眉最适合名媛气质，自然又有高级感
- 眉色应与发色协调，偏浅更温柔

---

## 四、唇妆篇

### 1. 豆沙色系
**关键词**：mauve lipstick, dusty rose lips, nude pink lipstick
```
elegant mauve lipstick, dusty rose lip color, sophisticated nude-pink lips, matte mauve lip
```

### 2. 玫瑰色系
**关键词**：rose lipstick, rosy pink lips, romantic lip color
```
romantic rose lipstick, soft rose-pink lips, elegant berry rose lip color
```

### 3. 咬唇妆
**关键词**：gradient lips, ombre lip makeup, Korean gradient lips
```
soft gradient lip makeup, natural ombre lips, inner lip color with soft edges, blurred lip line
```

### 实用技巧
- 名媛唇妆以"饱满不张扬"为主
- 可添加"glossy lips"或"satin finish"描述光泽感

---

## 五、腮红篇

### 位置与颜色
**关键词**：soft blush, natural flush, peachy cheeks
```
delicate peach blush on cheekbones, soft pink flush on apples of cheeks, natural rosy glow, subtle blush on high cheekbones
```

### 实用技巧
- 腮红位置：苹果肌向太阳穴方向晕染
- 颜色选择：蜜桃色、珊瑚粉、玫瑰粉
- 强调"自然红晕感"而非明显色块

---

## 六、修容篇

### 高光
**关键词**：subtle highlighter, natural glow, luminous highlight
```
delicate highlighter on cheekbones, subtle glow on nose bridge, luminous highlight on cupid's bow
```

### 阴影
**关键词**：soft contour, natural shadow, subtle face sculpting
```
soft contour on cheekbones, subtle jawline definition, natural nose contour
```

### 实用技巧
- 名媛修容讲究"自然立体"，避免过度修容
- 高光强调：颧骨、鼻梁、眉骨、唇峰
- 阴影强调：颧骨下方、下颌线、鼻侧

---

## 八、整体妆容风格篇

### 1. 韩式名媛妆
**关键词**：Korean beauty, K-beauty makeup, Korean elegant makeup
```
Korean-style elegant makeup, soft K-beauty look with dewy skin, refined Korean makeup aesthetic
```

**特点**：水光肌、平眉、粉色系眼妆、咬唇妆

### 2. 法式名媛妆
**关键词**：French beauty, Parisian makeup, French elegant style
```
French elegant makeup, Parisian chic beauty, effortless French glamour, sophisticated French style
```

**特点**：自然底妆、野生眉、红唇或裸唇、简约眼妆

### 3. 千金妆/富家女妆
**关键词**：rich girl makeup, wealthy aesthetic, luxurious makeup
```
expensive-looking makeup, luxurious beauty aesthetic, refined wealthy girl makeup, high-end glamorous look
```

**特点**：精致全妆、大地色眼妆、玫瑰豆沙唇、完美肤质

### 4. 轻欧美名媛妆
**关键词**：soft glam, elegant western makeup, refined glam look
```
soft glam makeup, elegant western beauty, refined glamorous look, sophisticated makeup with defined features
```

**特点**：立体修容、挑眉、大地色眼妆、饱满唇妆
