## Usage with OpenClaw

### Interactive CLI

```bash
# Analyze Bitcoin on Binance (1-day timeframe, markdown output)
openclaw skills/trading-analyzer analyze.py BTCUSDT --exchange BINANCE --timeframe 1D

# Analyze Apple stock (default markdown output)
openclaw skills/trading-analyzer analyze.py AAPL

# Get JSON output for integration
openclaw skills/trading-analyzer analyze.py AAPL --output json
```

### MCP Integration (Python)

```python
from analyze import AssetDetector, CryptoAnalyzer, StockAnalyzer, ReportGenerator

# Detect asset type
asset_type = AssetDetector.detect("BTCUSDT")  # Returns: "crypto"

# Analyze crypto
analysis = CryptoAnalyzer.analyze("BTCUSDT", exchange="BINANCE", timeframe="1D")
report = ReportGenerator.markdown_report("BTCUSDT", analysis, "crypto")
print(report)

# Analyze stock
analysis = StockAnalyzer.analyze("AAPL")
report = ReportGenerator.json_report("AAPL", analysis, "stock")
print(report)
```

## Report Components Reference

### Crypto Reports Include

- **Price Overview**: Current price, 24h change, high/low, volume
- **Technical Analysis**: Trend, RSI, MACD, Bollinger Bands, support/resistance levels
- **Market Sentiment**: Overall bullish/bearish assessment
- **Recommendation**: BUY/SELL/HOLD with risk level

### Stock Reports Include

- **Price Overview**: Current price, daily change, open, high/low, volume
- **Fundamentals**: P/E ratio, market cap, dividend yield, growth metrics
- **Latest News**: Top 3 recent news items with sentiment
- **Earnings**: Next earnings date, previous and estimated EPS
- **Recommendation**: BUY/SELL/HOLD with target price and risk level
