# Supported Model Repositories

## Printables
- **URL**: https://printables.com
- **Best for**: High-quality community models, free and paid
- **Features**: Advanced search, reviews, remix support
- **API**: Web scraping (no official API)
- **File formats**: STL, 3MF

## MakerWorld (Bambu Lab Official)
- **URL**: https://makerworld.com
- **Best for**: Models optimized for Bambu Lab printers
- **Features**: Official Bambu support, bench test results
- **API**: Web scraping (no official API)
- **File formats**: 3MF (native), STL

## Thingiverse
- **URL**: https://thingiverse.com
- **Best for**: Large community library, established models
- **Features**: Advanced search, community feedback
- **API**: REST API available (unofficial documentation)
- **File formats**: STL, SCAD, other

## MyMiniFactory
- **URL**: https://myminifactory.com
- **Best for**: Curated, high-quality models
- **Features**: Quality checks, remix support
- **API**: Web scraping (no official API)
- **File formats**: STL, 3MF, OBJ

## Search Tips

### By Model Type
- **Dragons**: Search "dragon", "wyvern", "drake"
- **Chess**: Search "chess piece", "chess set"
- **Miniatures**: Search "miniature", "tabletop", "D&D"
- **Functional**: Search "organizer", "holder", "stand"

### By Complexity
- **Beginner**: Filter by "easy print", "quick print" (<2 hours)
- **Intermediate**: Medium complexity, multi-color support
- **Advanced**: Requires supports, calibration, post-processing

### By Print Time (A1 Mini)
- **Quick** (< 2 hours): Small pieces, decorative items
- **Medium** (2-8 hours): Medium models, functional prints
- **Long** (> 8 hours): Large models, split into parts if needed
