# 财务工作流程与模板

## 经费收支管理流程

### 1. 预算编制流程
```
年度预算编制 → 部门预算申报 → 财务审核 → 管理层审批 → 预算下达 → 预算执行监控
```

**预算模板要素**：
- 收入预算：主营业务收入、政府补助、投资收益等
- 成本预算：直接成本、间接成本、研发费用、销售费用、管理费用
- 现金流预算：经营现金流、投资现金流、筹资现金流
- 资本性支出预算：设备采购、软件开发、办公场所装修等

### 2. 费用报销流程
```
员工提交报销申请 → 部门负责人审批 → 财务初审 → 财务复核 → 付款处理 → 凭证归档
```

**报销单必备要素**：
- 报销人信息、部门、日期
- 费用明细（日期、事由、金额、发票号）
- 审批人签字（部门负责人、财务负责人）
- 付款方式（现金、银行转账、支付宝/微信）
- 附件清单（发票、合同、验收单等）

### 3. 合同付款流程
```
合同签订 → 付款申请 → 财务审核 → 管理层审批 → 付款执行 → 合同归档
```

**合同付款审核要点**：
- 合同有效性（签署、盖章、授权）
- 付款条件满足情况（验收、发票、质保等）
- 付款金额与合同一致性
- 税务合规性（发票类型、税率、开票信息）

## 税务申报流程

### 增值税申报流程
```
1. 收集进项发票（认证勾选）
2. 汇总销项发票数据
3. 计算应纳税额
4. 填写增值税申报表
5. 网上申报
6. 缴纳税款
7. 保存申报资料
```

### 企业所得税预缴流程
```
1. 计算当期利润总额
2. 调整纳税调整事项
3. 计算应纳税所得额
4. 计算应纳所得税额
5. 填写预缴申报表
6. 网上申报缴纳
```

## 年报和审计流程

### 年度审计准备清单
- **财务资料**：
  - 全年会计凭证、账簿、报表
  - 银行对账单、银行存款余额调节表
  - 应收应付账款明细及账龄分析
  - 固定资产清单及折旧计算表
  - 存货盘点表及计价方法说明

- **业务资料**：
  - 全年合同台账（销售、采购、租赁等）
  - 重要会议决议（董事会、股东会）
  - 公司章程、营业执照等证照
  - 知识产权证书、资质证书
  - 政府补助文件及相关协议

- **税务资料**：
  - 全年纳税申报表及缴税凭证
  - 税务稽查、检查相关文件
  - 税收优惠备案文件
  - 关联交易相关资料

### 年报披露要点
- 财务报表附注披露要求
- 关联方交易披露
- 或有事项披露
- 重要会计政策变更说明
- 期后事项说明

### 银行账户管理
- **账户分类**：基本户、一般户、专用户、外币户
- **对账频率**：每月至少一次银行对账
- **印鉴管理**：财务章、法人章分开保管
- **网银管理**：操作员、复核员权限分离

## 内部控制要点

### 财务审批权限
- **费用报销**：部门经理≤5000元，财务总监≤20000元，总经理>20000元
- **合同付款**：按合同金额分级审批
- **资金调拨**：大额资金调拨需双人复核

### 职责分离原则
- 会计与出纳职责分离
- 采购与付款职责分离
- 销售与收款职责分离
- 资产保管与记录职责分离
