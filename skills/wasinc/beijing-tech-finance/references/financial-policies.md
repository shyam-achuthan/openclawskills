# 北京市科技公司财务管理制度

## 1. 会计制度基础

### 1.1 会计准则适用
- **小企业会计准则**：适用于年营业额≤5000万元的科技公司
- **企业会计准则**：适用于规模较大或有融资需求的科技公司
- **研发费用核算**：严格按照《企业会计准则第6号——无形资产》执行

### 1.2 会计科目设置
```
资产类：
- 1001 库存现金
- 1002 银行存款  
- 1122 应收账款
- 1403 原材料（研发耗材）
- 1601 固定资产
- 1701 无形资产（软件、专利等）

负债类：
- 2202 应付账款
- 2211 应付职工薪酬
- 2221 应交税费

权益类：
- 4001 实收资本
- 4101 资本公积
- 4103 本年利润

成本类：
- 5001 生产成本（研发项目成本）
- 5101 制造费用

损益类：
- 6601 销售费用
- 6602 管理费用
- 6603 财务费用
- 6711 营业外支出
- 6301 营业外收入
```

## 2. 经费收支管理

### 2.1 收入管理
- **主营业务收入**：技术服务、软件销售、咨询收入
- **其他业务收入**：培训收入、维护收入
- **营业外收入**：政府补助、投资收益

### 2.2 成本费用管理
- **研发费用**：人员工资、材料费、设备折旧、外包费用
- **管理费用**：办公费、差旅费、招待费、房租
- **销售费用**：市场推广、客户维护

### 2.3 审批权限
| 金额范围 | 审批人 | 备注 |
|---------|--------|------|
| ≤5,000元 | 部门经理 | 日常运营支出 |
| 5,001-20,000元 | 财务总监 | 重要支出 |
| 20,001-100,000元 | 总经理 | 重大支出 |
| >100,000元 | 董事会 | 战略性支出 |

## 3. 报销管理制度

### 3.1 报销原则
- **真实性**：所有报销必须有真实业务背景
- **合规性**：符合税法和公司制度要求
- **及时性**：费用发生后30天内报销

### 3.2 报销单据要求
- **增值税专用发票**：优先索取，用于进项税抵扣
- **增值税普通发票**：无法取得专票时使用
- **收据**：仅限小额零星支出（≤500元）
- **电子发票**：需提供PDF原件，防止重复报销

### 3.3 报销流程
1. 员工填写报销单 → 2. 部门经理审核 → 3. 财务初审 → 4. 财务总监审批 → 5. 出纳付款

## 4. 税务管理

### 4.1 主要税种
- **增值税**：软件企业可享受即征即退政策（实际税负超过3%部分退还）
- **企业所得税**：高新技术企业享受15%优惠税率
- **个人所得税**：按月代扣代缴
- **印花税**：合同、账簿等应税凭证

### 4.2 税收优惠政策
- **高新技术企业认定**：15%企业所得税率
- **软件企业双软认定**：增值税即征即退
- **研发费用加计扣除**：100%加计扣除（制造业）或75%（其他）
- **小微企业优惠**：年应纳税所得额≤300万元，分段计算

### 4.3 税务申报周期
- **增值税**：按月或按季申报
- **企业所得税**：按季预缴，年度汇算清缴
- **个人所得税**：按月申报
- **印花税**：按次或按期申报

## 5. 年报和审计

### 5.1 年度报告要求
- **工商年报**：每年1月1日-6月30日
- **税务年报**：次年5月31日前完成汇算清缴
- **统计年报**：根据统计局要求按时填报

### 5.2 审计要求
- **一般企业**：可选择是否进行审计
- **高新技术企业**：建议进行专项审计
- **融资企业**：投资人通常要求年度审计

## 6. 财务总监职责

### 6.1 核心职责
- 制定和完善财务管理制度
- 编制年度财务预算和资金计划
- 监督日常财务收支和会计核算
- 税务筹划和合规管理
- 财务分析和经营决策支持
- 融资和投资者关系管理

### 6.2 关键绩效指标
- 现金流管理：确保公司正常运营
- 成本控制：研发费用占比合理
- 税务优化：合法降低税负
- 财务合规：零重大违规事件

## 7. 科创企业特殊考虑

### 7.1 研发费用管理
- **人员费用**：研发人员工资、社保、公积金
- **直接投入**：材料、燃料、动力费用
- **折旧摊销**：研发设备、软件摊销
- **设计试验**：新产品设计、工艺规程制定
- **委托研发**：委托外部机构研发费用

### 7.2 股权激励会计处理
- **股份支付**：按公允价值确认费用
- **等待期**：分期确认管理费用
- **行权**：增加实收资本和资本公积

### 7.3 融资相关财务处理
- **天使轮/VC轮**：计入实收资本和资本公积
- **可转债**：按金融工具准则处理
- **政府补助**：与资产相关或与收益相关分别处理
