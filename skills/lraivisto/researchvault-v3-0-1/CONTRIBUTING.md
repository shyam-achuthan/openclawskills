# Contributing to ResearchVault

Welcome! We appreciate your help in making ResearchVault better.

## How to Contribute

1.  **Report Bugs**: Open an issue on GitHub.
2.  **Suggest Features**: Open an issue to discuss.
3.  **Submit Pull Requests**:
    *   Fork the repository.
    *   Create a feature branch.
    *   Ensure all tests pass (`pytest`).
    *   Submit a PR with a clear description of changes.

## Security Considerations

Since this tool handles web ingestion, please prioritize SSRF safety and data scrubbing in your contributions.
