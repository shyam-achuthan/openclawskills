# Personal Love

Помощник по отношениям для OpenClaw. Помнит контекст, важные даты, помогает разобраться в ситуациях.

## Что умеет

- Запоминать статус и контекст отношений
- Трекать важные даты (годовщина, ДР партнёра)
- Помогать с идеями для свиданий и подарков
- Поддерживать и помогать разобраться в ситуации
- Хранить важные факты о партнёре

## Команды

```
статус: в отношениях с [имя]   — записать статус
годовщина [дата]               — записать важную дату
др партнёра [дата]             — записать день рождения
запомни: [факт]                — сохранить важное
идеи для свидания              — предложить идеи
идеи подарка                   — помочь с подарком
хочу поговорить                — режим поддержки
инструкция                     — показать все команды
```

## Установка

```bash
clawdhub install personal-love
```

## Данные

Хранятся в `knowledge/personal/love.md` в воркспейсе агента.
